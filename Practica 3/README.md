## Practica 3 - Contador hasta/desde 100
###Objetivo
El objetivo de esta práctica era realizar, por medio de Verilog, un sistema capaz de contar incremental o decrementalmente en un rango de 0 a 100. Además, el contador tenía que ser capaz de cargar un número y contar a partir de ahí.
